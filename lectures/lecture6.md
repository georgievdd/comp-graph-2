# Лекция 6. Геометрические преобразования. Методы интерполяции.

## Геометрические преобразования

- Кадрирование (cropping) — вырезание прямоугольного фрагмента; изменение соотношения сторон (aspect ratio)
- Масштабирование
- Поворот
- Зеркальное отражение
- Скос
- **Аффинные преобразования** — параллельные прямые → параллельные прямые
- Коррекция геометрических искажений (дисторсии)
- Переформатирование (retargeting)

---

## Однородные координаты

Аффинные преобразования задаются матрицей 3×3:
```
T = [a   b   0]
    [c   d   0]
    [tx  ty  1]
```

Два способа записи:
1. `[x' y' 1] = [x y 1] × T`  — вектор-строка умножается на T
2. `[x' y' 1]ᵀ = Tᵀ × [x y 1]ᵀ`  — Tᵀ умножается на вектор-столбец

```
x' = ax + cy + tx
y' = bx + dy + ty
```

Если выполняется несколько преобразований T₁, T₂, T₃:
- Способ 1: `T = T₁T₂T₃`
- Способ 2: `T = T₃ᵀT₂ᵀT₁ᵀ`

Матричное перемножение некоммутативно → важен порядок.

---

## Общая схема преобразования

1. Угловые пиксели исходного изображения умножаются на **прямую** матрицу T → координаты угловых пикселей и размер нового изображения.
2. Координаты каждого пикселя нового изображения умножаются на **обратную** матрицу T⁻¹ → координата соответствующего пикселя в исходном.
3. Если координата попала в пределы исходного, значение определяют интерполяцией по соседним пикселям.

---

## Масштабирование (Scaling)

Прямая матрица:
```
[Sx  0   0]
[0   Sy  0]
[0   0   1]
```

Обратная матрица:
```
[1/Sx  0    0]
[0     1/Sy 0]
[0     0    1]
```

---

## Поворот (1) — вокруг начала координат

Поворот на угол φ **против** часовой стрелки:

Прямая:
```
[ cosφ   sinφ  0]
[-sinφ   cosφ  0]
[ 0      0     1]
```

Обратная:
```
[cosφ  -sinφ  0]
[sinφ   cosφ  0]
[0      0     1]
```

Повороты на углы, кратные 90°, реализуют как частные случаи (быстрее).

---

## Поворот (2) — вокруг центра изображения

Прямая матрица = произведение трёх матриц:
```
[1   0   0]   [cosφ   sinφ  0]   [1   0   0]
[0   1   0] × [-sinφ  cosφ  0] × [0   1   0]
[-xc -yc 1]   [0      0     1]   [xc  yc  1]
```
(перенос в (xc,yc)) × (поворот) × (перенос обратно)

**In-place поворот** (в том же буфере, in situ) = 2 последовательных скоса (shear):
```
[cosφ  -sinφ  0]   [cosφ  0  0]   [1   -tanφ  0]
[sinφ   cosφ  0] = [sinφ  1  0] × [0    secφ  0]
[0      0     1]   [0     0  1]   [0    0      1]
```

---

## Зеркальное отражение (Flip)

Относительно вертикальной оси:
```
T = [-1  0  0]
    [ 0  1  0]
    [ 0  0  1]
```

Относительно горизонтальной оси:
```
T = [1   0  0]
    [0  -1  0]
    [0   0  1]
```

Обычно реализуют как частные случаи (без матричного умножения).

---

## Скос (Skew / Shear)

Скос по X:
```
T = [1     0  0]
    [tgφ   1  0]
    [0     0  1]
```

Скос по Y — транспонирование матрицы. Возможен скос одновременно по X и Y.

---

## Передискретизация (Resampling)

Передискретизация — изменение частоты дискретизации сигнала (числа отсчётов):
- ЦАП (интерполяция) → непрерывный сигнал → АЦП с новой частотой
- Уменьшение числа отсчётов = **децимация** (downsampling)
- Для изображений: upsizing/downsampling = masштабирование

---

## Способы интерполяции изображений

Два подхода:
1. **1D интерполяция** сначала по строкам, затем по столбцам
2. **2D интерполяция** двумерными функциями:
   - Ближайший сосед (nearest neighbour)
   - Билинейная (bilinear)
   - Бикубическая (bicubic)

---

## Способы 1D интерполяции

1. Функции:
   - 0-го порядка (по ближайшему отсчёту)
   - 1-го порядка (кусочно-линейная)
   - 2-го порядка (квадратичная, полиномы 2-й степени)
   - 3-го порядка (кубическая, сплайны)
2. Фильтры по критерию близости АЧХ к идеальному ФНЧ:
   - Sinc интерполяция
   - Фильтры Чебышева

---

## Sinc интерполяция (формула Уиттекера — Шеннона)

```
x(t) = Σ_{n=-∞}^{∞} x[n] · sinc((t - nT) / T)
```
где T — период дискретизации.

**Ограничения применимости:**
1. Результат Фурье x(t) для частот больше B равен 0.
2. Частота дискретизации ≥ 2B, т.е. 1/(2B) > T.

**Нормированная** функция sinc(x):
```
sinc(x) = sin(πx)/(πx),  x ≠ 0
sinc(0) = 1
```

**Ненормированная** (математика): `sinc(x) = sin(x)/x`

### C++ реализация

```cpp
#include <math.h>

double sinc(double x)
{
    return (x == 0. ? 1. : sin(M_PI * x) / (M_PI * x));
}

void SincInterpolation1D(const double* pbIn, int lSizeIn,
        double* pbOut, int lSizeOut)
{
    double st = (double)(lSizeIn - 1) / (lSizeOut - 1);
    for (int i = 0; i < lSizeOut; ++i)
    {
        double sum = 0.;
        for (int j = 0; j < lSizeIn; ++j)
        {
            sum += pbIn[j] * sinc(st * i - j);
        }
        pbOut[i] = sum;
    }
}
```

## Фильтр Ланцоша (Lánczos)

Sinc фильтр при увеличении даёт ringing («звон»). Ланцош применил sinc в оконной функции:

```
S(x) = Σ_{i=⌊x⌋-a+1}^{⌊x⌋+a} sᵢ · L(x - i)
```

```
L(x) = { 1,                              если x = 0
        { a·sin(πx)·sin(πx/a)/(π²x²),   если -a ≤ x < a и x ≠ 0
        { 0,                              иначе
```

На практике `a = 2` или `a = 3`.  
Изображения интерполируют сначала по строкам, затем по столбцам.

---

## Интерполяция по ближайшему соседу

Значения в вершинах «единичного квадрата»:
```
f(0,0) = I([c], [r])
f(1,0) = I([c]+1, [r])
f(0,1) = I([c], [r]+1)
f(1,1) = I([c]+1, [r]+1)
```

```
I'(r,c) = f([c+0.5]-[c], [r+0.5]-[r])
```
где `[]` — операция взятия целой части.

---

## Билинейная интерполяция

Интерполяция по билинейной поверхности — 4 ближайших пикселя (3 одномерные интерполяции 1-го порядка):

```
I'(r,c) = f(0,0)(1-x)(1-y) + f(1,0)x(1-y) + f(0,1)(1-x)y + f(1,1)xy
```
где `x = c - [c]`, `y = r - [r]`

### C++ реализация масштабирования

```cpp
void ResizeBilinear(const unsigned char* pbIn, int lWidthIn,
        int lHeightIn, unsigned char* pbOut, int lWidthOut, int lHeightOut)
{
    for (int i = 0; i < lHeightOut; ++i)
    {
        double yy = (double)i * (double)lHeightIn / lHeightOut;
        int y = (int)yy;           // целая часть
        double u = yy - (double)y; // дробная часть
        for (int j = 0; j < lWidthOut; ++j)
        {
            double xx = (double)j * (double)lWidthIn / lWidthOut;
            int x = (int)xx;           // целая часть
            double v = xx - (double)x; // дробная часть
            // значения соседних пикселей
            int lP00 = pbIn[y * lWidthIn + x],
                lP01 = pbIn[y * lWidthIn + x + 1],
                lP10 = pbIn[(y+1) * lWidthIn + x],
                lP11 = pbIn[(y+1) * lWidthIn + x + 1];
            // билинейная интерполяция
            pbOut[i * lWidthOut + j] = (unsigned char)(
                (1. - u) * (1. - v) * lP00 +
                u  * (1. - v) * lP10 +
                v  * (1. - u) * lP01 +
                u  * v        * lP11);
        }
    }
}
```

## Бикубическая интерполяция

Интерполяция по **16** соседним точкам:

```
f(-1,-1)  f(0,-1)  f(1,-1)  f(2,-1)
f(-1, 0)  f(0, 0)  f(1, 0)  f(2, 0)
f(-1, 1)  f(0, 1)  f(1, 1)  f(2, 1)
f(-1, 2)  f(0, 2)  f(1, 2)  f(2, 2)
```

```
I'(c,r) = Σ_{k=1}^{16} bₖ · f(...)
```

Коэффициенты (x = c - [c], y = r - [r]):
```
b₁  = (x-1)(x-2)(x+1)(y-1)(y-2)(y+1)/4
b₂  = -x(x-2)(x+1)(y-1)(y-2)(y+1)/4
b₃  = -(x-1)(x-2)(x+1)y(y-2)(y+1)/4
b₄  = x(x-2)(x+1)y(y-2)(y+1)/4
b₅  = -x(x-1)(x-2)(y-1)(y-2)(y+1)/12
b₆  = -(x-1)(x-2)(x+1)y(y-1)(y-2)/12
b₇  = x(x-1)(x-2)y(y-2)(y+1)/12
b₈  = x(x-2)(x+1)y(y-1)(y-2)/12
b₉  = x(x-1)(x+1)(y-1)(y-2)(y+1)/12
b₁₀ = (x-1)(x-2)(x+1)y(y-1)(y+1)/12
b₁₁ = x(x-1)(x-2)y(y-1)(y-2)/36
b₁₂ = -x(x-1)(x+1)y(y-2)(y+1)/12
b₁₃ = -x(x-2)(x+1)y(y-1)(y+1)/12
b₁₄ = -x(x-1)(x+1)y(y-1)(y-2)/36
b₁₅ = -x(x-1)(x-2)y(y-1)(y+1)/36
b₁₆ = x(x-1)(x+1)y(y-1)(y+1)/36
```

## Кубическая свёртка (Cubic Convolution)

Вариант бикубической интерполяции: 5 одномерных кубических интерполяций (4 по строкам + 1 по интерполированным значениям):

```
f(x) = f(-1)u(x-1) + f(0)u(x) + f(1)u(x+1) + f(2)u(x+2)
```

```
u(s) = { (a+2)|s|³ - (a+3)|s|² + 1,   0 < |s| ≤ 1
        { a|s|³ - 5a|s|² + 8a|s| - 4a, 1 < |s| ≤ 2
        { 0,                             2 < |s|
```

Параметр `a = -0.5` (автор алгоритма). Встречаются `a = -0.75` и `a = -1`.

## Сравнение методов интерполяции

| Метод | Скорость | Качество |
|-------|----------|---------|
| Ближайший сосед | Быстрее всего | Ниже |
| Билинейная | Средняя | Средняя |
| Бикубическая | Медленнее | Выше |

## Коррекция дисторсии

Геометрические искажения корректируются на основе данных **калибровки**:
1. Снимают калибровочное изображение с известными точками (например, решётка)
2. Для каждого треугольника (разбиение плоскости) определяют матрицу трансформации из системы линейных уравнений по 3 точкам:

```
[x'ᵢ  y'ᵢ  1] = [xᵢ  yᵢ  1] × T,   i = 0,1,2
```

Коэффициенты a, b, c, d, tx, ty вычисляются через детерминанты.  
Аналогично можно **исказить** изображение (warping).

---

## Переформатирование (Retargeting)

**Retargeting** — визуально незаметное изменение/добавление локальных фрагментов для изменения размера (соотношения сторон).

**Seam carving** — наиболее известный алгоритм:
1. ФВЧ строит карту «стоимости/энергии» каждого пикселя
2. Динамическим программированием находят **шов** (цепочку пикселей между противоположными сторонами) с **минимальной** стоимостью
3. Шов либо **удаляют** (изображение уменьшается), либо вдоль него **добавляют** новую цепочку (изображение увеличивается)
4. Итерируют до нужного размера
