# Компьютерная графика — Лабораторные работы

## Лабораторная работа №1 — Анализ изображений по яркости

### Описание

Программа выполняет статистический анализ изображения на основе яркостной составляющей (luminance по формуле BT.601).

### Реализованный функционал

- **Гистограмма яркостей** — распределение пикселей по уровням яркости (0–255)
- **Среднее, дисперсия, СКО** — основные статистические характеристики
- **Квартили** (Q1, медиана, Q3) — описание распределения яркости
- **Энтропия** — мера информативности изображения (в битах)
- **Энергия** — сумма квадратов вероятностей яркостей
- **Коэффициент асимметрии (skewness)** — смещение распределения относительно среднего
- **Эксцесс (kurtosis)** — «остроконечность» распределения
- **GLCM** (матрица совместной встречаемости) — текстурный анализ для горизонтального и диагонального смещений
- **Добавление гауссова шума (AWGN)** — зашумление изображения с заданной дисперсией
- **PSNR** — оценка качества зашумлённого изображения

### Тестовые изображения

| Файл | Описание |
