=== ВЫВОДЫ: СРАВНЕНИЕ ФИЛЬТРОВ ===

1. ФИЛЬТРАЦИЯ ГАУССОВОГО ШУМА (σ²=400)
   Исходный шум ≈ 18–19 дБ PSNR (зависит от изображения).

   Box 3×3 (≈24–31 дБ):
     Простой усредняющий фильтр. Быстрый. На сложных текстурных изображениях
     (baboon) даёт лучший результат, чем Box 7×7, т.к. мелкие детали не размыты.
     На гладких изображениях (gradient) заметно уступает Box 7×7.

   Box 7×7 (≈23–39 дБ):
     Усиленное размытие. Хорошо работает на плавных изображениях (gradient ≈39 дБ),
     но на богатых деталями изображениях (baboon ≈23 дБ) размывает структуру.

   Гауссов фильтр σ=2 (≈24–35 дБ):
     Взвешенное среднее с гауссовым ядром — лучший баланс: центр весит больше.
     Превосходит Box на большинстве изображений. Рекомендован как стандартный НЧ-фильтр.

   Оптимальный sigma (подбор по PSNR):
     Даёт значительный выигрыш (gradient: σ=5.00 → 45.39 дБ vs 38.78 дБ для σ=2).
     Для текстурных изображений оптимальный sigma мал (baboon: σ=0.75, иначе размытие).
     Вывод: sigma должен соответствовать уровню и типу шума, а не быть фиксированным.

   Билатеральный фильтр (≈25–31 дБ):
     Лучший результат на детальных изображениях (baboon: 25.99 vs 24.76 дБ у Box 3×3).
     На плавных изображениях проигрывает Box 7×7 (gradient: 31 vs 39 дБ), т.к.
     весовая функция диапазона мало улучшает уже однородные области.
     Ключевое преимущество: сглаживает шум ВНУТРИ регионов, не размывая границы.

2. ФИЛЬТРАЦИЯ ИМПУЛЬСНОГО ШУМА (10% соль/перец)
   Box 3×3 (≈22–25 дБ): уменьшает импульсный шум, но не устраняет полностью.
   Box 7×7 (≈22–29 дБ): лучше для импульсного шума на гладких изображениях.
   Гауссов σ=2 (≈23–30 дБ): близок к Box 7×7, хорош для крупных изображений.
   Пороговый t=50 (≈15 дБ — ХУЖЕ исходного шума!):
     Неожиданно плохой результат. Причина: если центральный пиксель сам является
     импульсным шумом (0 или 255), порог исключает большинство соседей и фильтр
     вырождается. Пороговое усреднение применимо, только если центр не зашумлён.
     Для импульсного шума нужен медианный фильтр (нелинейный, не реализован).

3. ДЕТЕКТИРОВАНИЕ ГРАНИЦ (ФВЧ)
   Лапласиан (4-связный / 8-связный):
     Вторая производная. Быстрый и прост. Реагирует на ВСЕ резкие изменения,
     включая шум → при малейшем зашумлении выход засорён ложными откликами.
     Применяется только на чистых изображениях.

   LoG (Laplacian of Gaussian), σ=2:
     Сначала гауссово размытие подавляет шум, затем лапласиан выделяет края.
     Значительно устойчивее к шуму, чем простой лапласиан.
     Нулевые пересечения (zero-crossing) дают тонкие замкнутые контуры.

   Морфологический градиент (дилатация − эрозия):
     Нелинейный ФВЧ. Вычисляет локальный диапазон яркостей.
     Устойчив к мелкому шуму (не применяет операций усреднения).
     Даёт более широкие контуры, чем LoG, но лучше выявляет крупные структуры.
     Размер структурирующего элемента управляет масштабом детектируемых объектов.

4. СВОДНАЯ ТАБЛИЦА РЕКОМЕНДАЦИЙ
   +------------------------------+------------------------------------------+
   | Задача                       | Рекомендуемый фильтр                     |
   +------------------------------+------------------------------------------+
   | Гауссов шум, детальн. изобр. | Билатеральный или Гауссов (подобр. σ)    |
   | Гауссов шум, гладк. изобр.   | Box 7×7 или Гауссов (большой σ)          |
   | Импульсный шум (соль/перец)  | Медианный (нелинейный); Box как паллиатив|
   | Детектирование границ        | LoG + zero-crossing (тонкие замкн. края) |
   | Грубые края / объекты        | Морфологический градиент                 |
   | Повышение резкости           | Unsharp masking (на предв. сглаженном)   |
   | Быстрая обработка            | Box через интегральное изображение O(1)  |
   +------------------------------+------------------------------------------+
