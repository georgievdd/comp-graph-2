ДЗ: улучшение видимости слабоконтрастного текста
Изображение: 9.tif (Fri May 08 02:48:26 MSK 2026)

ЗАДАЧА: Изображение содержит слабоконтрастный светлый текст на тёмном
фоне И тёмный текст на светлом фоне. Глобальные методы (линейная TRC,
выравнивание гистограммы) улучшают один тип контраста за счёт другого.
Требуется ЛОКАЛЬНЫЙ метод, адаптирующийся к каждому участку.

КРИТЕРИЙ КАЧЕСТВА
  Среднее значение локального контраста Михельсона по тайлам 32×32:
  Q = mean_tile{ (max-min)/(max+min+1) }  ∈ [0,1], выше = лучше.
  Тайл 32×32 пкс соответствует размеру нескольких символов текста.
  Побочно: глобальное СКО (не критерий, для информации).

  Исходное:  1452×1070, mean=98,5, std=121,5
РЕЗУЛЬТАТЫ КРИТЕРИЯ КАЧЕСТВА (Q = контраст Михельсона, tile=32×32)

  Метод                           Q (↑лучше)   Глоб. СКО
  ----------------------------------------------------------
  Оригинал                        0,5155       121,5
  1. Эквализация гистограммы      0,1849       62,6
  2. CLAHE (tile=64, clip=3)      0,5201       110,8
  2. CLAHE (tile=64, clip=6)      0,5160       107,6
  3. Unsharp (σ=3, k=2, T=3)      0,5433       120,3
  3. Unsharp (σ=5, k=3, T=5)      0,5614       119,1
  4. Retinex SSR (σ=40)           0,2177       42,7
  5. Сигмоида (tile=64, T=20)     0,5140       121,3
  6. Вычитание фона (σ=60)        0,1187       29,5

АНАЛИЗ МЕТОДОВ (из лекции 7)

1. Эквализация гистограммы (глобальная)
   Улучшает общий динамический диапазон. Но CDF одна на всё
   изображение: если в нём есть и тёмные, и светлые области,
   одни участки улучшаются, другие — ухудшаются.

2. CLAHE (адаптивная эквализация по тайлам) — ЛУЧШИЙ МЕТОД
   Для каждого тайла строится своя TRC на основе локальной
   гистограммы. Ограничение clip предотвращает усиление шума.
   Билинейная интерполяция TRC между тайлами устраняет блочные
   артефакты. Обрабатывает оба типа контраста независимо.
   clip=3 — умеренное усиление; clip=6 — агрессивное.

3. Нерезкое маскирование
   Усиливает высокие частоты: I' = I + k*(I - Gauss(I)).
   Повышает резкость границ текста, но не устраняет неравномерный
   фон. При большом k появляются ореолы (halo) вдоль границ.

4. Ретинекс SSR
   I' = log(I+1) - log(LPF(I)+1). Моделирует восприятие яркости
   человеком (константность яркости). Хорошо подавляет градиент
   освещённости, но снижает яркостный масштаб.

5. Фильтр уменьшения длины перепада (сигмоида)
   Для каждого тайла: L = mean(25% минимальных), H = mean(25% максимальных).
   Нормализует в [0,1], применяет сигмоиду y=x²/(x²+(1-x)²), возвращает в [L,H].
   Растягивает серединную зону яркостей в тайле — усиливает локальный контраст.

6. Вычитание фона
   I' = (I - Gauss(I, σ_large)) + 128. Убирает низкочастотный
   тренд освещённости. Аналог HPF — хорошо для текста на
   неравномерном фоне, но теряет общий тон изображения.

ВЫВОД: Нерезкое маскирование (σ=5, k=3) даёт наибольший Q=0.56 согласно критерию.
Оно усиливает высокочастотные перепады (границы текст/фон), что напрямую
повышает локальный контраст в тайлах 32×32 вокруг символов.

CLAHE (clip=3) — второй по эффективности и предпочтительнее при наличии
шума или ореолов: ограничение clip предотвращает артефакты при агрессивном
усилении. Для изображения с обоими типами контраста (светлый текст на тёмном
и тёмный на светлом) CLAHE адаптируется к каждому участку независимо.

Глобальные методы (histEq, bgSubtract, Retinex) снижают Q, т.к. перераспределяют
контраст глобально и «выравнивают» локальные различия, ухудшая читаемость текста.
